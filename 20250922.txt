Специальные биты управления доступом
1. userbit - утсанавливается только для файлов
Позволяет запустить файл от имени владельца необходим для программ выполняющихся от имени обычных пользователей владельцем, для программ которые изменяют обьекты файловой системы защищенные от записи для всех пользователей
2. Установка userbit:
В режиме администратора:
udo chmod u+s programname
sudo chmod u+s passwd
3. Вывод stat для программы passwd:
 stat /bin/passwd
  Файл: /bin/passwd
  Размер: 64152         Блоков: 128        Блок В/В: 4096   обычный файл
Устройство: 8/4 Инода: 3150126     Ссылки: 1
Доступ: (4755/-rwsr-xr-x)  Uid: (    0/    root)   Gid: (    0/    root)
4. В случае установки с бита владельца для файла у которого небыло прав доступа на выполнение с бит выводится в верхнем регистре:
 stat /bin/passwd
  Файл: /bin/passwd
  Размер: 64152         Блоков: 128        Блок В/В: 4096   обычный файл
Устройство: 8/4 Инода: 3150126     Ссылки: 1
Доступ: (4664/-rwSrw-r--)  Uid: (    0/    root)   Gid: (    0/    root)
5. Сброс userbit:
sudo chmod u-s programname
