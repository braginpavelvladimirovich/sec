Установка группового бита на файлы и на папки
sudo chmod g+s file
Установка г-бита на файл позволяет запускать от имени группы
Установка г-бита на папку позволяет наследовать группу владельца на обьекты расположенные внутри
Пример прав доступа к файлу с г-битом:
2664/рв-рвСр--
Пример прав доступа к папке с г-битом:
2775/рвхрвср-х
Установка стики-бита на папку:
sudo chmod o+t dir
По умолчанию файлы созданные владельцем внтри папки могут удалять владельцы или группа владельцев
Папки с установленным т битом запрещают удаление всем кроме владельца при этом могут  его редактировать
Смена владельцев в группе:
sudo chown suername:groupname filename/directoryname
mkdir dir/dir/dir/dir создает папку в папке с группой владельца и наследует г-бит
touch dir/file создает файл с группой владельца г-бит не наследуется файлом
sudo chmod o+t
Пример прав доступа к папке с т-битом:
1775/рвхрвхр-т
При одновременной установки г и т-битов на папку, наследование распространяется только на г-бит
